# Crypto Exchange Widget

A modern, multi-step crypto-to-fiat exchange interface built with **Next.js**, **TypeScript**, and **shadcn/ui**. This project implements a "Wizard" style form that handles currency conversion inputs, payment method selection, and recipient details.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui

## Setup Instructions

### 1. Clone the repository

git clone https://github.com/couch-p0tato/novacrust-assessment.git

### 2. Install Dependencies

npm install
# or
yarn install
# or
pnpm install

### 3. Run the development server

npm run dev

Open http://localhost:3000 with your browser to see the result.
